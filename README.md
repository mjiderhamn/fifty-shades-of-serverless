# Fifty shades of Serverless

Demo code for [a talk on Serverless](http://java.jiderhamn.se/2018/11/22/fifty-shades-of-serverless/) demonstrating
* AWS Lambda
  * HTTP/API Gateway
  * SQS trigger
  * S3 trigger
* AWS Lambda using Jersey
* Azure Functions
  * HTTP
  * Queue trigger
  * Blob storage trigger
* Spring Cloud Function
